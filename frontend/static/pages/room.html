Jak czytać chorwacki?

1. Czytasz tak jak widzisz
Chorwacki jest bardzo fonetyczny — prawie każda litera ma jeden dźwięk.
2. J = jak polskie "j"
np. ja, ljubav → czytasz normalnie jak w polskim
3. C = „c”, Č = „cz”, Ć = miękkie „ć”
cesta = cesta
čaj = czaj
kuća = kuća (miękkie ć)
4. Š = sz, Ž = ż/rz
škola = szkoła
žena = żena
5. Dž i Đ
dž = dż (jak „dżem”)
đ = coś między „dź” a „dż” (trzeba się osłuchać)
6. Lj i Nj to pojedyncze dźwięki
ljubav = „lubav” (jednym dźwiękiem)
konj = „koń”
7. Akcent zwykle na pierwszą sylabę
DO-bar, KU-ća, ŠKO-la
